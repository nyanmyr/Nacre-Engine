Nacre Engine is a Game Engine with an Entity-Component-System (ESC) architecture.

Built with SFML (2.6.1).

Currently Features:
- Components
- Entity Manager
- ComponentManager (Registry)
- Systems
	- Input
	- Movement
	- Rendering (Shapes)
